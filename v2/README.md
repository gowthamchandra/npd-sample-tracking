# Sample Tracking v2

Fresh rebuild with:
- Username/password login (fixed demo users)
- Role-based access control (Admin / Quality / Logistics / Marketing)
- SQLite storage
- No external dependencies

## Run

```bash
python3 v2/server.py
```

Open `http://127.0.0.1:8010`.

## Demo Users

- `admin / Admin@123`
- `quality / Quality@123`
- `logistics / Logistics@123`
- `marketing / Marketing@123`
